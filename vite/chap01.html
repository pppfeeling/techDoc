Chapter 1: Vite로 React 앱 빌드하기

2025.05.27 | React + Vite 완전정복

🎯 Vite가 뭐길래?

Vite(비트)는 프랑스어로 "빠르다"는 뜻! ⚡ 개발 서버 시작이 번개처럼 빠르고, 핫 리로드가 즉시 반영되는 차세대 빌드 도구입니다.

🐌 기존 방식 (CRA)
  • 개발 서버 시작: 30-60초 😴
  • 번들링 기반 (Webpack)
  • 모든 파일을 미리 번들링
  • HMR 속도 느림
npx create-react-app my-app cd my-app npm start # 느린 시작...
⚡ Vite 방식
  • 개발 서버 시작: 1-3초 🚀
  • ES 모듈 기반
  • 필요한 파일만 즉시 제공
  • 초고속 HMR
npm create vite@latest my-react-app -- --template react cd my-react-app npm install npm run dev # 빠른 시작!
1 프로젝트 생성

다양한 템플릿으로 프로젝트를 즉시 생성할 수 있어요!

# React 기본 템플릿 npm create vite@latest # TypeScript 템플릿 npm create vite@latest my-app -- --template react-ts
2 개발 서버 실행

설치 후 바로 개발 서버를 실행하면 끝!

cd my-app npm install npm run dev # ➜ Local: http://localhost:5173/
3 실제 경험해보기

파일을 수정하고 저장하면 즉시 반영! 🎉

// App.jsx 수정 후 저장 function App() { return ( <h1>안녕 Vite! 🚀</h1> ) }

🔥 Vite의 핵심 비밀

개발 환경에서:

  • ES 모듈을 브라우저에 직접 제공
  • 변경된 모듈만 갱신 (HMR)
  • 번들링 없이 즉시 실행

프로덕션 빌드:

  • Rollup으로 최적화된 번들 생성
  • 트리 쉐이킹 자동 적용
  • 코드 스플리팅 지원
