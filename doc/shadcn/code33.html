Resizable Panels (크기 조절 가능 패널)

📜개요

Resizable Panels 기능은 사용자가 레이아웃 내의 패널들 사이의 경계선을 드래그하여 각 패널의 크기를 동적으로 조절할 수 있게 합니다. 📐 이는 복잡한 정보를 여러 영역에 나누어 보여주고, 사용자가 필요에 따라 각 영역의 크기를 최적화할 수 있도록 하여 유연한 레이아웃 경험을 제공합니다.

주요 사용 사례:

  • IDE의 코드 편집기, 파일 탐색기, 터미널 등의 레이아웃
  • 메일 클라이언트의 메일 목록, 메일 내용 보기 패널
  • 대시보드에서 여러 차트나 위젯 영역의 크기 조절
  • 문서 편집기의 사이드바와 메인 콘텐츠 영역

Shadcn UI는 이 기능을 위해 react-resizable-panels 라이브러리를 사용하며, 이를 위한 스타일링된 컴포넌트 (ResizablePanelGroup, ResizablePanel, ResizableHandle)를 제공합니다.

✨ Resizable Panels는 사용자에게 레이아웃 제어권을 부여하여 개인화된 작업 환경을 만들 수 있게 해줍니다!

⚙️기본 설정 및 설치

먼저 react-resizable-panels 라이브러리를 설치해야 합니다:


npm install react-resizable-panels
# 또는
yarn add react-resizable-panels

그리고 shadcn UI에서 제공하는 Resizable 관련 컴포넌트들 (ResizablePanelGroup, ResizablePanel, ResizableHandle)을 CLI를 통해 프로젝트에 추가합니다.

🧩주요 컴포넌트 및 역할

Shadcn UI의 Resizable 기능은 세 가지 주요 컴포넌트로 구성됩니다:

  • <ResizablePanelGroup> (루트): 크기 조절이 가능한 패널들의 전체 그룹 컨테이너입니다. 패널들의 정렬 방향(수평/수직)과 전체적인 동작을 제어합니다.
    • 주요 Props: direction ("horizontal" | "vertical"), autoSaveId (레이아웃 상태 자동 저장 ID), onLayout (레이아웃 변경 시 콜백), className 등.
  • <ResizablePanel>: 그룹 내의 개별 패널입니다. 각 패널은 초기 크기, 최소/최대 크기 등을 가질 수 있습니다.
    • 주요 Props: defaultSize (초기 크기, % 단위 숫자), minSize, maxSize, collapsible (접기 가능 여부), collapsedSize (접혔을 때 크기), onCollapse, onExpand, order (패널 순서), id (패널 식별자), className 등.
  • <ResizableHandle>: 두 ResizablePanel 사이에 위치하여 사용자가 드래그하여 크기를 조절할 수 있는 핸들(경계선)입니다.
    • 주요 Props: withHandle (핸들 내부에 시각적 요소를 추가할지 여부), disabled, id, onDragging (드래그 중 콜백), className 등.

💡사용 예제 (핵심 코드)

다음은 수평 및 수직으로 크기 조절이 가능한 패널 레이아웃 예제입니다.


import {
  ResizablePanelGroup,
  ResizablePanel,
  ResizableHandle,
} from "@/components/ui/resizable" // shadcn UI 컴포넌트

export function ResizableLayoutDemo() {
  return (
    <ResizablePanelGroup
      direction="horizontal" // 수평으로 패널 배치
      className="max-w-full h-[500px] rounded-lg border" // 전체 크기 및 스타일
      onLayout={(sizes: number[]) => { // 레이아웃 변경 시 크기 값 (비율)
        console.log("Layout changed (horizontal):", sizes)
      }}
    >
      <ResizablePanel defaultSize={25} minSize={15}>
        <div className="flex h-full items-center justify-center p-6">
          <span className="font-semibold">사이드바</span>
        </div>
      </ResizablePanel>
      <ResizableHandle withHandle /> {/* 핸들 중앙에 아이콘 표시 */}
      <ResizablePanel defaultSize={75}>
        <ResizablePanelGroup direction="vertical" onLayout={(sizes: number[]) => {
          console.log("Layout changed (vertical):", sizes)
        }}>
          <ResizablePanel defaultSize={25} minSize={20}>
            <div className="flex h-full items-center justify-center p-6">
              <span className="font-semibold">헤더 영역</span>
            </div>
          </ResizablePanel>
          <ResizableHandle /> {/* 아이콘 없이 기본 핸들 */}
          <ResizablePanel defaultSize={75}>
            <div className="flex h-full items-center justify-center p-6">
              <span className="font-semibold">메인 콘텐츠</span>
            </div>
          </ResizablePanel>
        </ResizablePanelGroup>
      </ResizablePanel>
    </ResizablePanelGroup>
  )
}

export default ResizableLayoutDemo;
💡 팁: ResizablePanel의 defaultSize는 백분율(0-100 사이 숫자)로 지정합니다. 그룹 내 패널들의 defaultSize 합이 100이 되도록 설정하는 것이 일반적입니다. minSize와 maxSize를 사용하여 각 패널의 최소/최대 크기를 제한할 수 있습니다. ResizablePanelGroup은 중첩하여 복잡한 다단 레이아웃도 구성할 수 있습니다. autoSaveId prop을 사용하면 사용자가 조절한 레이아웃 크기를 로컬 스토리지에 자동으로 저장하고 다음에 불러올 수 있습니다!
