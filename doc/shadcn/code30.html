Carousel (캐러셀)

📜개요

Carousel(또는 슬라이더)은 여러 개의 콘텐츠(이미지, 카드, 텍스트 블록 등)를 가로 또는 세로로 순환하며 보여주는 UI 컴포넌트입니다. 🔄 사용자는 네비게이션 버튼(이전/다음), 도트 인디케이터, 또는 드래그(터치)를 통해 슬라이드를 탐색할 수 있습니다.

주요 특징:

  • 동적인 콘텐츠 표시: 여러 항목을 한 공간에 효과적으로 보여줌
  • 다양한 네비게이션 옵션: 버튼, 도트, 드래그, 자동 재생 등
  • 반응형 디자인: 화면 크기에 따라 슬라이드 개수나 레이아웃 조절 가능
  • 플러그인 시스템 (embla-carousel): 자동 재생, 페이드 효과 등 다양한 기능 확장 가능

Shadcn UI는 embla-carousel-react라는 가볍고 유연한 캐러셀 라이브러리를 기반으로 스타일링된 예제를 제공합니다. embla-carousel-react는 핵심 로직만 제공하는 헤드리스 UI에 가까워, UI 구조와 스타일링은 사용자가 직접 구성해야 합니다. Shadcn UI는 이를 위한 기본 마크업과 스타일을 제공합니다.

✨ Carousel은 콘텐츠를 매력적으로 보여주고 사용자 참여를 유도하는 데 효과적인 방법입니다!

⚙️기본 설정 및 설치

먼저 embla-carousel-react 라이브러리를 설치해야 합니다:


npm install embla-carousel-react
# 또는
yarn add embla-carousel-react

Shadcn UI에서 Carousel 예제를 사용하려면, 관련 컴포넌트(Carousel, CarouselContent, CarouselItem, CarouselPrevious, CarouselNext, CarouselApi 타입)를 CLI를 통해 프로젝트에 추가합니다.

🔗핵심 개념 및 컴포넌트

Shadcn UI의 Carousel은 embla-carousel-react 훅과 Context API를 사용하여 구성됩니다. 주요 컴포넌트와 역할은 다음과 같습니다:

  • <Carousel> (루트): 캐러셀의 전체 컨테이너이자 Context Provider입니다. embla-carousel-react의 useEmblaCarousel 훅을 내부적으로 사용하며, 캐러셀 API와 상태를 하위 컴포넌트에 제공합니다.
    • 주요 Props: opts (embla 옵션), plugins (embla 플러그인), orientation ("horizontal" | "vertical"), setApi (API 인스턴스 콜백)
  • <CarouselContent>: 실제 슬라이드 아이템들을 감싸는 뷰포트(viewport) 역할을 합니다. 이 요소에 overflow: hidden이 적용됩니다.
  • <CarouselItem>: 개별 슬라이드 항목입니다. 이 안에 이미지, 카드 등 실제 콘텐츠가 들어갑니다.
  • <CarouselPrevious>: 이전 슬라이드로 이동하는 버튼입니다. (shadcn Button 기반)
  • <CarouselNext>: 다음 슬라이드로 이동하는 버튼입니다. (shadcn Button 기반)

useCarousel 훅을 사용하면 <Carousel> 컴포넌트 외부에서도 캐러셀 API에 접근하여 제어할 수 있습니다 (예: 커스텀 도트 인디케이터 구현).

🛠️Embla Carousel 옵션 및 플러그인

embla-carousel-react의 useEmblaCarousel 훅은 다양한 옵션을 제공하여 캐러셀의 동작을 세밀하게 제어할 수 있습니다. 이 옵션들은 <Carousel opts={...} /> prop을 통해 전달됩니다.

  • align: 슬라이드 정렬 방식 ("start", "center", "end" 또는 숫자).
  • loop: 무한 루프 여부 (boolean).
  • startIndex: 시작 슬라이드 인덱스 (number).
  • draggable: 마우스/터치 드래그 가능 여부 (boolean).
  • slidesToScroll: 한 번 스크롤 시 이동할 슬라이드 개수 (number 또는 "auto").
  • ... 등 다양한 옵션이 있습니다. (공식 문서 참고)

또한, 플러그인을 사용하여 기본 기능 외의 동작을 추가할 수 있습니다. <Carousel plugins={[...]} /> prop을 사용합니다.

  • Autoplay: 자동 재생 기능.
  • ClassNames: 특정 상태에 따라 클래스 추가.
  • WheelGestures: 마우스 휠 제스처 지원.
  • 사용자 정의 플러그인도 생성 가능.

💡사용 예제 (핵심 코드)

다음은 기본적인 이미지 캐러셀 예제입니다. (shadcn/ui 공식 문서 예제 기반)


"use client"

import * as React from "react"
import Autoplay from "embla-carousel-autoplay" // 자동 재생 플러그인 예시

import { Card, CardContent } from "@/components/ui/card" // 슬라이드 내용 예시
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,
  type CarouselApi, // API 타입
} from "@/components/ui/carousel"
import { Button } from "@/components/ui/button" // 네비게이션 버튼 스타일링용

export function ImageCarousel() {
  const [api, setApi] = React.useState<CarouselApi>()
  const [current, setCurrent] = React.useState(0)
  const [count, setCount] = React.useState(0)

  React.useEffect(() => {
    if (!api) {
      return
    }
    setCount(api.scrollSnapList().length)
    setCurrent(api.selectedScrollSnap() + 1)

    api.on("select", () => {
      setCurrent(api.selectedScrollSnap() + 1)
    })
  }, [api])

  const plugin = React.useRef( // 자동 재생 플러그인 설정
    Autoplay({ delay: 2000, stopOnInteraction: true })
  )

  return (
    <div className="w-full max-w-xs mx-auto"> {/* 캐러셀 전체 크기 제어 */}
      <Carousel
        opts={{
          align: "start",
          loop: true,
        }}
        plugins={[plugin.current]} // 플러그인 적용
        setApi={setApi} // API 인스턴스 받기
        className="w-full"
        onMouseEnter={plugin.current.stop} // 마우스 올리면 자동 재생 멈춤
        onMouseLeave={plugin.current.reset} // 마우스 벗어나면 자동 재생 다시 시작
      >
        <CarouselContent>
          {Array.from({ length: 5 }).map((_, index) => (
            <CarouselItem key={index} className="md:basis-1/2 lg:basis-1/3"> {/* 반응형 슬라이드 크기 */}
              <div className="p-1">
                <Card>
                  <CardContent className="flex aspect-square items-center justify-center p-6 bg-muted">
                    <span className="text-3xl font-semibold">{index + 1}</span>
                  </CardContent>
                </Card>
              </div>
            </CarouselItem>
          ))}
        </CarouselContent>
        <CarouselPrevious className="absolute left-[-50px] top-1/2 -translate-y-1/2" /> {/* 버튼 위치 커스텀 */}
        <CarouselNext className="absolute right-[-50px] top-1/2 -translate-y-1/2" />
      </Carousel>
      <div className="py-2 text-center text-sm text-muted-foreground">
        슬라이드 {current} / {count}
      </div>
      <div className="flex justify-center space-x-2 mt-2">
        {/* 커스텀 도트 인디케이터 예시 */}
        {Array.from({ length: count }).map((_, index) => (
          <Button
            key={index}
            variant={index + 1 === current ? "default" : "outline"}
            size="icon"
            className="h-2 w-2 rounded-full p-0" // 작은 원형 버튼
            onClick={() => api?.scrollTo(index)}
          />
        ))}
      </div>
    </div>
  )
}

export default ImageCarousel;
💡 팁: CarouselItem의 className에 Tailwind CSS의 basis 유틸리티(예: md:basis-1/2)를 사용하면 화면 크기에 따라 한 번에 보이는 슬라이드 개수를 조절할 수 있습니다. setApi 콜백으로 받은 API 인스턴스(api)를 사용하면 api.scrollTo(index), api.scrollPrev(), api.scrollNext() 등으로 캐러셀을 프로그래매틱하게 제어할 수 있어 커스텀 네비게이션(예: 도트 인디케이터) 구현이 용이합니다.
