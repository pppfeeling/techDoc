Date Picker & Calendar

📜개요

Shadcn UI에서 날짜 관련 기능은 주로 react-day-picker 라이브러리를 기반으로 한 Calendar 컴포넌트와, 이를 Popover와 결합하여 만든 Date Picker 형태로 제공됩니다. 📅

  • Calendar: 독립적으로 달력을 표시하고 날짜를 선택할 수 있게 하는 컴포넌트입니다.
  • Date Picker: 일반적으로 버튼 형태로 되어 있으며, 클릭 시 Popover 내에 Calendar가 나타나 날짜를 선택하게 하는 복합 컴포넌트입니다.

이 컴포넌트들은 단일 날짜 선택, 날짜 범위 선택, 특정 날짜 비활성화, 월/연도 이동 등 다양한 기능을 지원하며, shadcn UI의 스타일링을 통해 미려한 UI를 제공합니다.

✨ react-day-picker의 강력한 기능과 shadcn UI의 스타일이 만나 사용자 친화적인 날짜 선택 경험을 제공합니다!

⚙️기본 설정 및 설치

react-day-picker와 날짜 포맷팅을 위한 date-fns 라이브러리를 설치해야 합니다:


npm install react-day-picker date-fns
# 또는
yarn add react-day-picker date-fns

그리고 shadcn UI에서 제공하는 Calendar 컴포넌트와, Date Picker를 만들기 위해 Button, Popover 관련 컴포넌트들을 프로젝트에 추가합니다 (CLI 사용).

📅Calendar 컴포넌트

<Calendar /> 컴포넌트는 react-day-picker의 DayPicker 컴포넌트를 shadcn/ui 스타일로 래핑한 것입니다. react-day-picker의 대부분의 props를 그대로 사용할 수 있습니다.

주요 Props (react-day-picker 기반):

Prop 이름 타입 설명
mode "single" | "multiple" | "range" | "default" 선택 모드를 지정합니다.
  • single: 단일 날짜 선택
  • multiple: 여러 날짜 선택
  • range: 날짜 범위 선택
  • default: 날짜 선택 기능 없이 달력만 표시 (기본값)
selected Date | Date[] | { from: Date; to: Date } | undefined 현재 선택된 날짜(들)입니다. mode에 따라 타입이 달라집니다. (제어)
onSelect (date, selectedDate, activeModifiers, e) => void 날짜가 선택될 때 호출되는 콜백입니다. mode에 따라 인자가 다릅니다. (제어)
defaultMonth Date 초기에 표시될 달(month)입니다.
month Date 현재 표시된 달을 제어합니다. (제어)
onMonthChange (month: Date) => void 표시된 달이 변경될 때 호출됩니다. (제어)
disabled Matcher | Matcher[] | (date: Date) => boolean 비활성화할 날짜(들)를 지정합니다. (예: { before: new Date() }, [{ dayOfWeek: [0, 6] }])
fromDate, toDate Date 선택 가능한 최소/최대 날짜를 지정합니다.
numberOfMonths number 한 번에 표시할 달의 개수입니다. (기본값 1)
showOutsideDays boolean 이전/다음 달의 날짜를 현재 달력에 표시할지 여부입니다. (기본값 true)
locale Locale (from date-fns) 달력의 로케일(언어, 날짜 형식 등)을 지정합니다. (예: import { ko } from 'date-fns/locale')
className string Calendar 루트 요소에 적용할 CSS 클래스입니다.
classNames object (react-day-picker) 달력 내부 요소들에 대한 CSS 클래스맵입니다.
...기타 react-day-picker props footer, captionLayout, modifiers, modifiersClassNames 등 매우 다양한 props를 지원합니다.

🗓️Date Picker (복합 컴포넌트)

Date Picker는 일반적으로 Button을 트리거로 사용하고, 클릭 시 Popover 내부에 Calendar를 표시하는 형태로 구성됩니다. 사용자가 날짜를 선택하면 Popover가 닫히고 버튼에 선택된 날짜가 표시됩니다.

구성 요소:

  • Button: Date Picker를 여는 트리거 버튼. (shadcn/ui)
  • Popover, PopoverTrigger, PopoverContent: Calendar를 담는 팝오버. (shadcn/ui)
  • Calendar: 실제 날짜 선택 UI. (shadcn/ui, react-day-picker 기반)

이러한 조합은 shadcn UI 공식 문서의 Date Picker 예제에서 잘 보여주고 있습니다.

💡사용 예제 (핵심 코드)

1. 기본 Calendar 사용 예제


"use client"

import * as React from "react"
import { Calendar } from "@/components/ui/calendar"
import { ko } from "date-fns/locale" // 한국어 로케일

export function SimpleCalendar() {
  const [date, setDate] = React.useState<Date | undefined>(new Date())

  return (
    <Calendar
      mode="single"
      selected={date}
      onSelect={setDate}
      locale={ko} // 한국어 설정
      className="rounded-md border"
      // disabled={(day) => day.getDay() === 0 || day.getDay() === 6} // 주말 비활성화 예시
    />
  )
}

2. Date Picker (Popover와 Calendar 결합) 예제


"use client"

import * as React from "react"
import { format } from "date-fns"
import { ko } from "date-fns/locale"
import { Calendar as CalendarIcon } from "lucide-react"

import { cn } from "@/lib/utils"
import { Button } from "@/components/ui/button"
import { Calendar } from "@/components/ui/calendar"
import {
  Popover,
  PopoverContent,
  PopoverTrigger,
} from "@/components/ui/popover"

export function DatePickerDemo() {
  const [date, setDate] = React.useState<Date | undefined>()

  return (
    <Popover>
      <PopoverTrigger asChild>
        <Button
          variant={"outline"}
          className={cn(
            "w-[280px] justify-start text-left font-normal",
            !date && "text-muted-foreground"
          )}
        >
          <CalendarIcon className="mr-2 h-4 w-4" />
          {date ? format(date, "PPP", { locale: ko }) : <span>날짜를 선택하세요</span>}
        </Button>
      </PopoverTrigger>
      <PopoverContent className="w-auto p-0">
        <Calendar
          mode="single"
          selected={date}
          onSelect={(selectedDate) => {
            setDate(selectedDate)
            // Popover를 수동으로 닫고 싶다면 Popover의 open 상태를 관리해야 함
            // 여기서는 Calendar 선택 시 자동으로 닫히도록 Popover가 동작할 수 있음 (내부 로직)
          }}
          initialFocus // Popover가 열릴 때 Calendar에 자동 포커스
          locale={ko}
        />
      </PopoverContent>
    </Popover>
  )
}

3. Date Range Picker (날짜 범위 선택) 예제 (mode="range" 사용)


"use client"

import * as React from "react"
import { format } from "date-fns"
import { ko } from "date-fns/locale"
import { Calendar as CalendarIcon } from "lucide-react"
import { DateRange } from "react-day-picker" // DateRange 타입 import

import { cn } from "@/lib/utils"
import { Button } from "@/components/ui/button"
import { Calendar } from "@/components/ui/calendar"
import {
  Popover,
  PopoverContent,
  PopoverTrigger,
} from "@/components/ui/popover"

export function DateRangePickerDemo({
  className,
}: React.HTMLAttributes<HTMLDivElement>) {
  const [dateRange, setDateRange] = React.useState<DateRange | undefined>({
    from: new Date(2023, 11, 20), // 예시: 2023년 12월 20일
    to: new Date(2023, 11, 24),   // 예시: 2023년 12월 24일
  })

  return (
    <div className={cn("grid gap-2", className)}>
      <Popover>
        <PopoverTrigger asChild>
          <Button
            id="date"
            variant={"outline"}
            className={cn(
              "w-[300px] justify-start text-left font-normal",
              !dateRange && "text-muted-foreground"
            )}
          >
            <CalendarIcon className="mr-2 h-4 w-4" />
            {dateRange?.from ? (
              dateRange.to ? (
                <>
                  {format(dateRange.from, "LLL dd, y", { locale: ko })} -{" "}
                  {format(dateRange.to, "LLL dd, y", { locale: ko })}
                </>
              ) : (
                format(dateRange.from, "LLL dd, y", { locale: ko })
              )
            ) : (
              <span>기간을 선택하세요</span>
            )}
          </Button>
        </PopoverTrigger>
        <PopoverContent className="w-auto p-0" align="start">
          <Calendar
            initialFocus
            mode="range"
            defaultMonth={dateRange?.from}
            selected={dateRange}
            onSelect={setDateRange}
            numberOfMonths={2} // 두 개의 달을 함께 표시
            locale={ko}
          />
        </PopoverContent>
      </Popover>
    </div>
  )
}
💡 팁: date-fns 라이브러리를 사용하여 선택된 날짜를 원하는 형식으로 포맷팅할 수 있습니다 (예: format(date, "yyyy년 MM월 dd일")). react-day-picker는 매우 다양한 props를 제공하므로, 공식 문서를 참고하여 주말 비활성화, 특정 날짜 강조, 커스텀 셀 렌더링 등 고급 기능을 구현할 수 있습니다.
