Chapter 2: 정적 자산 다루기

2025.05.27 | Assets 완전정복

🎯 정적 자산이란?

이미지, 폰트, CSS, JSON 등 앱에서 사용하는 모든 파일들! Vite는 이들을 똑똑하게 처리합니다 ✨

📁 Vite 프로젝트 구조 이해하기
my-vite-app/ ├── public/ ← 🌍 직접 제공되는 정적 파일들 │ ├── favicon.ico │ ├── logo.png │ └── robots.txt ├── src/ ← 🔧 빌드 과정을 거치는 소스 파일들 │ ├── assets/ ← 🎨 처리되는 자산들 │ │ ├── images/ │ │ ├── styles/ │ │ └── fonts/ │ ├── components/ │ └── App.jsx └── index.html ← 📄 진입점 HTML

핵심 차이점: public/은 그대로 복사, src/assets/는 최적화 처리됩니다!

1 🌍 Public 폴더 방식

URL로 직접 접근 가능한 파일들

// public/logo.png 파일을 사용 function Header() { return ( <img src="/logo.png" alt="로고" /> ) } // ✅ 브라우저에서: /logo.png
  • 빌드 시 그대로 복사됨
  • URL 변경 없음
  • 큰 파일이나 외부 라이브러리에 적합
2 🔧 Import 방식

모듈로 가져와서 최적화되는 파일들

// src/assets/hero.jpg 파일을 사용 import heroImage from './assets/hero.jpg' function Hero() { return ( <img src={heroImage} alt="히어로 이미지" /> ) } // ✅ 빌드 후: /assets/hero-a1b2c3.jpg
  • 파일명에 해시 추가 (캐싱 최적화)
  • 이미지 압축 및 최적화
  • 사용하지 않는 파일 제거
🚀 실제 프로젝트 예제
// src/components/ProductCard.jsx import React from 'react' import './ProductCard.css' // CSS 파일 import import placeholderImg from '../assets/images/placeholder.png' function ProductCard({ product }) { return ( <div className="product-card"> <img src={product.image || placeholderImg} alt={product.name} className="product-image" /> <h3>{product.name}</h3> <p>{product.price}원</p> </div> ) } export default ProductCard
/* src/components/ProductCard.css */ .product-card { background: white; border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.1); overflow: hidden; transition: transform 0.2s; } .product-card:hover { transform: translateY(-4px); } .product-image { width: 100%; height: 200px; object-fit: cover; }

🎨 다양한 파일 형식 지원

파일 타입 지원 형식 특별 기능
이미지 png, jpg, gif, svg, webp 자동 최적화, 해시 네이밍
폰트 woff, woff2, ttf, otf CSS에서 직접 import 가능
스타일 css, scss, less, stylus CSS 모듈, PostCSS 자동 처리
데이터 json, txt, csv JSON은 트리 쉐이킹 지원
🎯 언제 Public을 쓸까?
  • favicon.ico - 루트에 있어야 함
  • robots.txt - SEO 파일
  • 외부 라이브러리 - CDN 대신
  • 큰 미디어 파일 - 비디오, 대용량 이미지

Public 파일은 절대경로 /로 시작해야 해요!

⚡ 언제 Import를 쓸까?
  • 컴포넌트 이미지 - 로고, 아이콘
  • CSS 파일 - 스타일시트
  • 작은 자산들 - 아이콘, 썸네일
  • 동적으로 선택되는 파일

Import된 파일은 빌드 시 최적화되어 성능이 좋아져요!
